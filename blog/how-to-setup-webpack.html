How To Setup Webpack

Here's a super simple list of procedures to get Webpack setup on a Mac computer.

  1. Open Terminal and cd to the working directory
  2. sudo npm install webpack -g (install Webpack globally)
  3. npm init
  4. npm install webpack --save-dev

If you don't do step #2 to install Webpack globally, then you won't be able to run the webpack command on the project to compile.
